# Reproducibility Package Slot

This directory stores DB reproducibility export packages generated by:

- `server/src/db/reproducibility-package.ts`
- `npm run repro:export` (from `Project-Memory-MCP/server`)

Expected package path:

- `database-seed-resources/reproducibility/db-repro-package.json`

Validation output (optional):

- `database-seed-resources/reproducibility/db-repro-parity-report.json`
